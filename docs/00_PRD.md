# PRD

**Purpose**
- Define the goal and scope of the performance meal planner.

**Inputs**
- User profile data (`schemas/user_profile.schema.json`)
- Weekly context data (`schemas/weekly_context.schema.json`)
- Outcome signals (`schemas/outcome_signals.schema.json`)

**Outputs**
- Weekly outputs package (`schemas/weekly_outputs.schema.json`)

**Owner/Process**
- Owner: Product lead
- Process: Weekly refinement based on outcomes and feedback

**Done when**
- Demo run produces valid weekly outputs and matches templates
